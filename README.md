# hello-github

GitHub 学习与测试仓库 / GitHub Learning & Testing Repository

## 简介 / About

这是用于测试 GitHub 基本使用方法和技巧的演练仓库。

This is a playground repository for practicing GitHub workflows, Git commands, and development patterns.

## 内容 / Contents

参考学习笔记 (`docs/shirp_github_note/`):

- Git 基础概念与分支管理
- GitHub 仓库基础与高级功能
- GitHub Desktop / IDEA / VS Code 集成
- GitHub Actions (CI/CD)
- GitHub Pages 部署
- Webhooks 与 REST API
- GitHub Packages
- Git LFS (大文件存储)
- 安全功能

## 快速开始 / Quick Start

```bash
# 查看仓库状态
git status

# 查看提交历史
git log --oneline

# 创建新分支
git checkout -b feature/test

# 合并分支
git merge feature/test
```
